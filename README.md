# Customer Shopping Behavior Analysis

## Overview

This project presents an end-to-end data analytics solution focused on understanding customer shopping behavior. Using Python, MySQL, Power BI, Excel, and PowerPoint, the project transforms raw transaction data into actionable business insights through data cleaning, exploratory analysis, SQL querying, dashboard visualization, and business storytelling.
The objective is to identify purchasing trends, customer segments, product preferences, and subscription behavior to support data-driven business decisions.

## Business Problem

Retail businesses generate large volumes of customer transaction data, but valuable insights often remain hidden within the data.

This project aims to answer key business questions such as:

* Which customer groups generate the most revenue?
* How do subscriptions influence spending behavior?
* Which products perform best across categories?
* What impact do discounts have on purchasing patterns?
* Which customer segments should be targeted for growth?

---
## Dataset

The dataset contains customer transaction records including:

* Customer demographics
* Product information
* Purchase amounts
* Subscription status
* Discounts and promotions
* Shipping preferences
* Product ratings

**Dataset Size:** 3,900 customer transactions
